# 设计美学技能包

## 触发规则
在编写任何前端/UI代码前，必须先选择一个设计风格锚点，并根据该风格的系统令牌（配色/字体/间距/圆角/阴影/动效）来生成代码。

## 设计令牌模板
选定风格后, 必须输出此令牌系统的完整定义:
```
色彩: --bg, --surface, --text-primary, --text-secondary, --accent, --accent-glow
字体: --font-display, --font-body, --font-mono, 字号梯度
间距: --space-xs ~ --space-3xl
圆角: --radius-sm ~ --radius-full
阴影: --shadow-sm ~ --shadow-glow
动效: --ease-out, --duration-fast ~ slow
```

## 禁止模式
- 禁止 Inter + 紫色渐变 + 大圆角卡片 (AI味)
- 禁止未经风格声明的通用美观代码
- 禁止同一页面混合3种以上风格

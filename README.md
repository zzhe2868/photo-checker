# 照片废片检测工具

本地离线运行的AI照片质量检测工具，自动筛选三类废片：**模糊**、**人物闭眼**、**曝光异常**。

## 技术栈
- **OpenCV** — 模糊检测（拉普拉斯方差）+ 闭眼检测（Haar级联）
- **tkinter** — 桌面GUI界面
- **NumPy** — 直方图曝光分析

# 安装依赖
py -m pip install opencv-python numpy -i https://pypi.tuna.tsinghua.edu.cn/simple --timeout=100

# 运行
py photo_checker.py或双击 启动照片检测.bat

## 功能
1. 选择照片文件夹，批量扫描
2. 机械检测三类废片并显示原因
3. 实时进度条 + 废片统计
4. 一键将废片移动到 _废片/ 文件夹
5. 完全本地离线，不联网不上传

## 检测原理
| 类型 | 方法 | 判定标准 |
|------|------|----------|
| 模糊 | 拉普拉斯方差 | 方差 < 100 |
| 闭眼 | Haar级联人脸+人眼 | 人脸出现但眼睛 < 2 |
| 过曝/欠曝 | 直方图分析 | 过亮>80% 或 过暗>18% |
